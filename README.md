# Game Match API

API responsável pelo armazenamento de dados de jogos e pela lógica de recomendação do Game Match.

## Sobre o projeto

O objetivo da API é manter um catálogo de jogos e gerar recomendações personalizadas com base em um sistema de pesos definidos para cada jogo.

## Funcionalidades

- Armazenamento e gerenciamento de dados de jogos
- Lógica de recomendação baseada em pesos

## Tecnologias

- Go
- PostgreSQL
